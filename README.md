# 秋招简历自动填写助手

本项目是一个基于 Tampermonkey 的浏览器用户脚本，通过读取本地 `profile.json` 中的个人简历信息，在各大互联网公司招聘官网的在线表单中自动填写常见字段，减少重复手动输入。

---

## 功能特点

- 从本地 `profile.json` 中读取个人信息、教育经历、实习经历、语言能力、证书、技能、家庭情况等字段
- 自动填写常见表单字段：姓名、联系方式、证件号码、教育背景、实习经验、技能描述等
- 适配常见招聘网站的在线简历表单，包含但不限于：
  - 北森（beisen / zhiye.com 体系）
  - 猎聘、拉勾、智联招聘、前程无忧
  - BOSS 直聘
  - 美团、字节跳动、腾讯、阿里巴巴、华为、百度、京东、小米、滴滴、拼多多、网易、哔哩哔哩、米哈游等
- 针对 React 等受控组件表单做了专门适配，通过 label 文本、兄弟节点及父容器等多种策略定位输入框
- 所有数据仅存储在浏览器本地的油猴存储中，不会上传至任何服务器

---

## 使用前提

- 一款现代浏览器（如 Chrome、Edge 等）
- 已安装 Tampermonkey 或兼容的用户脚本管理扩展
- 本地准备好的 `profile.json` 文件（本仓库根目录提供了示例，可直接复制修改）

---

## 安装步骤

1. 在浏览器扩展商店中安装 Tampermonkey 扩展。
2. 打开 Tampermonkey 仪表盘，新建一个脚本。
3. 将本仓库中的 `resume_autofill.user.js` 文件全部内容复制粘贴到新脚本中并保存。
4. 确认脚本在目标招聘网站的域名下处于启用状态。

## 快捷操作与交互

- 页面右下角会出现一个浮动按钮（图标为剪贴板），点击后会弹出菜单，包含：
  - 自动填写当前页面表单
  - 导入 / 更新个人信息
  - 预览已保存的信息
  - 清除已保存的信息
- 键盘快捷键：
  - `Alt + F`：自动填写当前页面表单
  - `Alt + I`：打开“导入个人信息”面板
- 顶部会出现提示条用于显示成功、警告或信息提示，方便确认操作结果。

---

## 隐私与安全

- 本脚本只会从浏览器本地的油猴存储中读取和写入数据，不会向任何远程服务器发送个人信息。
- 建议在个人设备上使用本脚本，避免在公共或不受信任的环境中保存敏感信息。
- 如需在公共电脑上登录求职网站，使用完毕后请在菜单中清除已保存的信息，并退出浏览器账号。

---

## 已知限制

- 一些网站的下拉框（如性别、学历、政治面貌等）采用自定义组件实现，目前仍可能需要手动选择。
- 如招聘网站对页面结构或表单字段进行了大幅调整，可能导致部分字段无法自动匹配，需要对脚本进行适配更新。
- 本脚本主要面向中文简历表单，对非中文环境的站点支持有限。
